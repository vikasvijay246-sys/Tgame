# Typing Master Game

A modern typing practice game designed for beginners to advanced users.  
Built to improve typing speed, accuracy, focus, and keyboard confidence through interactive gameplay.

---

## Features

- Beginner to Pro typing levels
- Real-time typing accuracy tracking
- Words per minute (WPM) calculation
- Mistake highlighting system
- Dark Mode / Light Mode
- Responsive mobile-friendly design
- Smooth typing experience
- Clean and minimal UI
- Timer-based typing challenges
- Instant result analysis

---

## Game Levels

### Beginner
- Easy words
- Slow typing pace
- Basic keyboard training

### Intermediate
- Medium-length words
- Faster typing sessions
- Accuracy improvement focus

### Advanced
- Complex words and sentences
- High-speed typing
- Pressure-based challenges

### Pro Mode
- Extreme speed tests
- Long paragraph typing
- Precision and consistency focus
